# BURUNDIARXIV WEB CLIENT

### Installation

- `git@github.com:burundiarxiv/burundiarxiv-web-client.git`
- `cd burundiarxiv-web-client`
- `yarn install` install dependencies

### Run in development
- `yarn dev`

### Run in production
- `yarn build`
- `yarn start`
